## Question 1 — Time Complexity Analysis *(5 marks)*

For each function below, give the growth of the running time.

**(a)**
```java
public static int f1(int N) {
    int x = 0;
    for (int i = 0; i < N; i++)
        x++;
    return x;
}
```

**(b)**
```java
public static int f2(int N) {
    int x = 0;
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < i; j++) {
            x++;
        }
    }
    return x;
}
```

**(c)**
```java
int sum = 0;
for (int i = 1; i < N; i *= 2) {
    for (int j = 0; j < N; j++)
        sum++;
}
```

**(d)**
```java
public static int f4(int N) {
    if (N == 0) return 0;
    return f4(N/2) + f1(N) + f4(N/2);
}
```

**(e)**
```java
public static int f7(int N) {
    if (N == 1) return 0;
    return 1 + f7(N/2);
}
```

## Question 6 — Load Balancing *(5 marks)*

- **Assign request to a random server.**

**(a)** *(1 mark)*  
Round Robin

**(b)** *(1 mark)*  
Assign to server with minimal load

**(c)** *(3 marks)*
