- Edge (A, C): Connects the Peatland Area with the Residential Area

**(a)** *(1 mark)*

**(b)** Given the Kruskal's Algorithm below:

```
KRUSKAL(G):
    A = {}                      # Initialize an empty set for the MST
    for each vertex v in G:
        MAKE-SET(v)             # Create disjoint sets for each vertex

    sort the edges of G in increasing order by weight
    for each edge (u, v) in G sorted by weight:
        if FIND-SET(u) ≠ FIND-SET(v):
            A = A ∪ {(u, v)}   # Add edge to the MST
            UNION(u, v)         # Merge the sets

    return A                    # Return the edges in the MST
```

Explain how Kruskal's Algorithm operates in determining the firebreaks to minimize the spread and impact of peatland fires in the area. Identify the selected edges for inclusion in the Minimum Spanning Tree (MST), ensuring that no cycles are formed by the following:

**iii.** *(6 marks)*
